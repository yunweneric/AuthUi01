# Flutter Pizza Shop

Love pizza?
Found this UI on FIgma community

## Setup guide

- Install and use [Fvm](https://fvm.app/) to manage your flutter version

- Specify your fvm version

  ```
  fvm use 3.xx
  ```

- Get your dependencies
  ```
   flutter pub get
  ```
